# AI Public Squares v0.1.1

AI Public Squares 的首個可驗證 npm 試行版。

## 本版包含

- `@adamchanadam/aps` npm package 已發佈，安裝後可用 `npx aps bridge-pack` 輸出 Bridge Pack fixture。
- `bridge-pack --role B` 可輸出 User B 角色版本。
- README、GitHub Pages 公開文檔與 CLI help 已改為已驗證的安裝方式：先 `npm install --save-dev @adamchanadam/aps`，再 `npx aps ...`。
- 全面檢已補齊本地 HTML 預覽、臨時 Hub 往返回歸、Bridge Pack 啟動行為記錄。

- 真實跨機運作仍待後續試行。
- 不建議用於正式生產或不可中斷的重要流程。
