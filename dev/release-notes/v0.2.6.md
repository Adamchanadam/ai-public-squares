# AI Public Squares v0.2.6 — Pre-release

This is a follow-up candidate after the 0.2.5 UAT upgrade test.

## Fixed

- `doctor` no longer frames the next step as only `inbox` / `publish`.
- `doctor` now points users first to the AI natural-language route and lists the full CLI fallback set.
- Remaining success messages for `inbox`, `consume`, `revise`, `withdraw`, and `close` are now in Traditional Chinese.
- `npx aps init` now uses a compact cat-to-cat APS Hub banner without side borders, reducing terminal alignment problems.
- Starter pack install wording now uses `npm install --save-dev @adamchanadam/aps@latest` for upgrade clarity.

## Updated

- README, public entry page, beginner walkthrough, skill runtime wording, and project index now reflect npm latest 0.2.6 and the 0.2.6 pre-release boundary.
